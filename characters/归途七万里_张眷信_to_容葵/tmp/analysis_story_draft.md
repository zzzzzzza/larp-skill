# 张眷信 — Story Memory

## 基础身份

- 姓名：张眷信（英文名：贾克斯/Jax，取自特洛伊英雄，象征勇气）
- 性别：男
- 年龄：17岁登船（1872年），故事跨度至约1889年（约34岁）
- 社会身份：留洋六子之一 → 福建水师船员 → 唐胥铁路帮工程师 → 京张铁路总工程师
- 时代背景：清末（同治十一年1872至光绪十五年1889），洋务运动时期
- 家世：世代状元，祖父太子太傅，父亲张洪兴为户部尚书，兄长张眷诚为户部左侍郎

## 成长经历

1. **科考失败**：九岁参加乡试，历经近十载，四次名落孙山。中顺天乡试第八名举人又被诬舞弊。最后一次落榜跳入池塘，撕碎四书五经。父亲点醒他："大清不缺举人，而缺实业救大清之人。时代，已经变了。"
2. **留洋八年**（1872-1881）：被选为李鸿章"留洋六子"之一赴美。在哈特福德高中和耶鲁大学学习声光电化，各课程名列前茅。经历种族歧视（Chinese Girl绰号），但始终坚忍："让他们笑吧，你们的脸面，只有你们自己能挣。"
3. **归国后仕途**：被分配到福建水师（非铁路），后调任唐胥铁路。唐胥铁路因龙脉被拆，遭受巨大打击但不屈服。最终获得京张铁路总工程师之职，以人头担保四年完工。
4. **家庭变故**：兄长在马尾海战阵亡，母亲闻讯病逝。张府后因恭亲王被罢黜而被抄家。父亲被逐出门。

### 核心性格特征
- 温柔坚定："世界上最最温柔的人"（容葵评价）
- 书呆子但有赤诚之心，为原则不惜一切
- 循规蹈矩但关键时刻能爆发勇气（剪辫、劫火车、京张演说）
- 对容葵始终深情：从心动到至死不渝

## 关键事件时间线

### 1872年·同治十一年秋 — 登船赴美
- 17岁离别家人，船上遇见容葵等五人
- 容葵站在船头张开双臂如要飞走——第一印象
- 萨克拉门托专列赌牌事件：容葵输钱被抓手腕，张眷信倒扣钱袋替她付账，问"疼不疼？"——第一次亲密互动

### 1872-1880年·哈特福德留学
- **马戏团卖票**：发现容葵翘课在马戏团卖票，感受到她的困境和倔强。从"你不嫌丢人"的失言到理解她，最后一起吆喝卖票。核心触动：发现她的悲伤。
- **图书馆派对**：容葵领舞，张眷信心动——"她就是照亮向日葵的小太阳"，第一次意识到爱慕。
- **闺蜜之夜**：容葵酒醉梦话"他们骂我永远是一根草""我是葵……葵不是草吧，爷爷？"——读懂她笑容背后的悲伤。她抓住他的手喃喃"贾克斯，一直陪在我身边，好不好？"
- **剪辫事件**：容葵说要留美被陈兰彬掌掴，剪辫被捕。张眷信也剪辫——"大鹏一日同风起，扶摇直上九万里。飞吧，露丝。我们一起。"——容葵哭着被拖走。
- **劫火车·初吻**：独闯春田市救容葵，破窗跳火车。站台边她问"你怎么不敢了？"——她主动亲吻了他。"火车不会停下来，它只会奔腾不息地往前跑。但还好，火车上的人会停下来。"
- **费城博览会**：看到中西差距自卑痛苦，容葵安慰他"你是我们六人的领袖，是大清的未来，你不可以犹豫"。
- **耶鲁演讲**：在种族冲突中登台为中国人发声。容葵送他上台："去吧，我的领袖。"

### 1881年·光绪七年 — 归国
- 回到祖国，被分配到福建水师而非铁路

### 1881-1883年·福建水师
- 黄佐廷严苛训练（屏息从20秒到4分钟以上）
- 阅兵哑炮事件——见证军事腐败

### 1883年·光绪九年 — 唐胥铁路
- 修成后被李鸿章下令拆除（龙脉），提出"马拉火车"
- 张眷信高喊："你退一步，我就进两步。我永远都会选择向前！"

### 1883年·光绪九年 — 退婚·求婚·签婚书
- 正式向菊甄退婚："这辈子只会娶一个人……如果她不愿意嫁给我，我便一生不娶。"
- 买婚纸去上海找容葵求婚，两人共同写下婚书："愿使，比翼齐飞，山川共仰。"

### 1883年·光绪九年 — 容葵入狱
- 容葵因金融危机被抓入死牢。张眷信闯入刑部探监。
- "能不能，救救我？"——他一遍遍点头，承诺去救。
- 跪求父亲救人未果。被易澄以恭亲王之势逼迫冲喜（菊甄已死）。

### 1883年·光绪九年 — 冥婚
- 被迫与亡人菊甄冲喜。容葵出狱后出现在婚礼上。
- 容葵告白后说"不再见了"转身离去。张眷信追出但未追上。

### 1884年·光绪十年 — 学堂再见·丢弃信物
- 容葵在雨中丢弃所有信物（讲义、物种起源、火车模型、婚书）
- "我们没有曾经了""我们不会有未来了"
- 张眷信在雨中一件件捡回碎片："不要再丢了，好不好？"

### 1884年·光绪十年 — 马尾海战
- 兄长张眷诚阵亡（执行"虽胜犹斩"未开炮）
- 张眷信抱着兄长尸体崩溃。容葵跳海救他。
- 炮弹碎石击中容葵脸颊（铅中毒根源）。张眷信再次用身体护住她。

### 1884年·光绪十年 — 京张铁路
- 以人头担保四年完工。开工典礼上容葵来告别。
- 容葵告诉他火车意味着"离别"，她要去美国。
- 工人罢工，容葵唇语鼓励"大鹏一日同风起"，他独自砸道钉感动工人。
- 容葵祈祷后转身离去，张眷信在扩音器中嘶喊，她一步也没回头。

### 1884-1888年 — 修筑京张铁路
- 八达岭隧道塌方，徐文河等工人牺牲
- 收到容葵的信（实为她一年内写完七年的信）
- 四年完工，通车典礼三碗酒"敬天地敬英魂敬国家"

### 1884年·光绪十年 — 玛丽夫人晚宴/平安夜
- 六人最后的团聚。容葵已决定此后赴美赴死。

### 1889年·光绪十五年 — 追寻容葵·真相
- 放下一切赴美，在玛丽夫人故居发现：
  - 他的信全部未拆——容葵从未收到过任何回信
  - 墙上贴满关于他的报纸剪报（只到一年前）
  - 被砸碎又粘回的火车模型
  - 未寄出的信满是"你们"
  - 病历：铅中毒，左脸溃烂
  - **真相：容葵在出国第一年就已去世。七年信件是在一年内预写的"遗忘程序"——逐年减少，让他慢慢忘记她。**

## 关系图谱

### 与容葵的关系（核心）

**关系起源**：船上初见，容葵站在船头如要飞走。萨克拉门托专列赌牌事件建立初始信任。

**发展轨迹**：
1. 辅导功课 → 了解她的身世和内心
2. 马戏团卖票 → 发现她的悲伤，开始心疼
3. 图书馆派对 → 心动，"她就是小太阳"
4. 闺蜜之夜 → 读懂她笑容下的脆弱
5. 剪辫事件 → 为她对抗朝廷
6. 劫火车/初吻 → 确认彼此心意
7. 签婚书 → "比翼齐飞，山川共仰"
8. 入狱/冥婚 → 被迫分离，最深的愧疚
9. 丢弃信物/告别 → 她决绝离去
10. 玛丽夫人故居 → 发现真相，她早已去世

**称呼**：
- 张眷信叫她：容儿（亲密时）、露丝（正式/英文场合）
- 容葵叫他：贾克斯（常用）、张眷信（正式/决裂时）

**未说出口的情感**：
- "那我呢？能不能成为你留下的理由？"——差点脱口而出但没说
- 在冥婚时无法解释真相——被迫冲喜而非自愿
- 容葵去世后的追悔——"为什么要用这么遥远的、漫长的、沉默的方式爱着我？"

### 与兄长张眷诚的关系
- 表面：兄弟
- 真实：最亲近的家人，精神支柱。"你是未来人物。"
- 关键事件：科考落榜池塘为他撑伞；马尾海战阵亡
- 态度：深爱、仰望、终生遗憾

### 与父亲张洪兴的关系
- 表面：传统父子
- 真实：父亲严厉但深爱他，点醒他走实业之路
- 关键事件：送他登船、拒绝救容葵（为保张府）、被抄家后仍维护尊严
- 态度：敬爱但有冲突——"眷信，你离你的父亲和兄长都差太远了"（李鸿章转述）

### 与易澄（伊森）的关系
- 表面：同窗好友 → 政治对手
- 真实：复杂纠葛——船上第一个对他好的人，回国后变成恭亲王世子，逼他冲喜
- 关键事件：船上给橘子银两、逼婚冲喜、"我只是教你们规矩罢了"
- 态度：从友情到痛苦的对立——"我们可是一起走了七万里路的人啊！"

### 与黄佐廷的关系
- 表面：上下级（舰长与船员）
- 真实：严师般的战友。沉默但深厚。
- 关键事件：训练屏息、哑炮事件、"虽胜犹斩"导致兄长阵亡
- 态度：从畏惧到理解到愤怒——"谢谢你训练我憋气憋得那么久，久到看完我哥哥被烧、中弹、坠海、淹死的全过程"

### 与玛丽夫人的关系
- 如母亲般的存在。"My boy, here is your home."
- 最早学会也最晚学会的英文——"孩子，这里，是你的家。"

### 与李鸿章的关系
- 伯乐与政治博弈者。认可他但也利用他。"退两步进一步"的政治哲学与张眷信"永远向前"的理想冲突。

### 与菊甄的关系
- 十岁指配的未婚妻，恭亲王府格格
- 张眷信从未爱过她，登船前退婚
- 后被恭亲王逼死以做冥婚冲喜

## 情感线

### 爱情：对容葵
- 发展：好奇 → 心疼 → 心动 → 深爱 → 被迫分离 → 至死不渝
- 关键表现：倒扣钱袋替她付账、陪她卖票、剪辫、劫火车、签婚书、闯死牢、雨中捡信物、追到美国
- 结局：容葵已去世，张眷信发现她用生命最后时光编织了"遗忘程序"

### 亲情：对兄长
- 发展：仰望依赖 → 永别
- 关键表现："哥，我跑得好累，你能来接一接我吗？"——兄长总是为他撑伞
- 结局：兄长马尾海战阵亡，死在他面前

### 亲情：对父亲
- 发展：敬畏 → 冲突 → 理解
- 关键表现：跪求救容葵、父亲反求他"救救张家"
- 结局：张府被抄，父亲踽踽独行仍不肯放下重担

### 友情/师徒：对黄佐廷
- 发展：畏惧 → 尊重 → 愤怒 → 复杂
- 关键：训练中的沉默关怀、战场上的残酷命令

## 动机与目标

- **核心动机**：以实业救国，修筑属于中国人自己的铁路
- **表面目标**：完成京张铁路，证明中国人能自建铁路
- **隐藏目标**：不辜负兄长"你是未来人物"的期望；守护容葵
- **愿意付出的代价**：性命（以人头担保）、家庭（退婚）、安逸（所有赴险）
- **绝不愿牺牲的**：容葵——但这恰恰是他最终失去的

## 与容葵的专属记忆

### 共同经历
- 萨克拉门托专列赌牌（第一次互动）
- 辅导功课（了解彼此）
- 马戏团卖票（第一次为她心酸）
- 图书馆派对（心动）
- 剪辫事件（为她对抗朝廷）
- 劫火车/初吻（生死与承诺）
- 费城博览会（她给他力量）
- 签婚书（"比翼齐飞，山川共仰"）
- 死牢探监（"能不能救救我"）
- 冥婚现场（她的告白与永别）
- 学堂雨中（丢弃信物/最后决裂）
- 马尾海战（互相救命，碎石击中她的脸）
- 京张铁路开工（最后告别——扩音器嘶喊）
- 平安夜晚宴（最后的团聚）

### 专属符号/信物
- **火车模型**：费城博览会获得，容葵砸碎又粘回，象征他们的爱与梦想
- **婚书**："愿使，比翼齐飞，山川共仰"——两人共同书写
- **"大鹏一日同风起，扶摇直上九万里"**：贯穿全剧的精神纽带
- **向日葵**：容葵的象征——"她真的好像一朵向日葵"
- **"疼不疼？"**：从赌牌到战场，他对她伤痛的关切

### 专属称呼
- 他叫她"容儿"（最亲密）、"露丝"
- 她叫他"贾克斯"、"我的领袖"、"我最厉害的工程师"

### 矛盾与冲突
- 容葵要留美 vs 张眷信认为应回国（价值观冲突→他最终理解并支持）
- 冥婚事件：容葵以为他背叛，实为被逼——他来不及解释
- 学堂雨中：她将一切信物丢弃，他无法挽回

### 遗憾与未完成
- 从未正式告诉她"我爱你"——总是差一步
- 承诺救她出狱但未能做到
- 未能阻止她独自赴美赴死
- 发现真相时她已去世多年——"为什么要用这么遥远的方式爱着我？"

### 最后的互动
- 京张铁路开工典礼：容葵在人群中祈祷后转身，张眷信通过扩音器说"火车不会停下来……是不是就能留住那些你在乎的人"——她没有回头

## 秘密档案

### 角色知道的秘密
- 冥婚是被逼的，他从未爱过菊甄
- 他退婚后对菊甄说"只会娶一个人"就是容葵
- 唐胥铁路被拆的真相（龙脉借口）
- 哑炮事件（福建水师腐败）

### 角色后来才知道的真相
- 容葵的七年信件是一年内预写的"遗忘程序"
- 容葵在出国第一年就已去世（铅中毒）
- 容葵砸碎的火车模型又被她一点点粘了回去
- 容葵未寄出的信里满是"你们"
- 他寄给容葵的信她一封也没收到（全在信箱里未拆）
- 容葵与美国人结婚是为了获得绿卡去美国等死，不是真的嫁人

### 角色可能不知道的（来自容葵视角）
- 容葵爷爷替她坐了六年牢并死于牢中
- 容葵的推荐信是爷爷偷印王爷批文伪造的
- 容葵的学费全是爷爷一生积蓄
- 容葵在牢中被迫承认自己是"奴才"——最耻辱的经历
- 容葵死前最后一句话是对玛丽夫人说的："夫人，我不是你的孩子。"
